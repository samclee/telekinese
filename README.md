# Telekinessball

A two-player game of telekinetic soccer! Use a single button to levitate and hurl soccer balls across the field, with the winner being the first to 10 points

## Controls
### Player 1
#### Keyboard
Movement - WASD
Action - space
#### Controller
Movement - left stick or d-pad
Action - A
Reset - start

### Player 2
#### Keyboard
Movement - arrow keys
Action - enter
#### Controller
Movement - left stick or d-pad
Action - A
Reset - start

### Assorted controls
Reset - r
Fullscreen - f
Quit - q or escape
